Sokoban

Keke eres tú.

Keke vive en una cueva llena de cajas.

Keke no tiene brazos (😿), así que no puede tirar de ellas, solo empujar una a la vez.

Keke quiere ordenarlas, y poner cada una sobre una bandera.

Cuando una caja está sobre una bandera se vuelve azul. ¡Magia!

Ordena todas las cajas de un nivel para completarlo y conseguir una estrella.


🧦 Pulsa las teclas de flechas (o WASD) para moverte.

⌛ Pulsa la tecla R para reiniciar el nivel actual, y Z para deshacer el último movimiento.

🔎 Usa el zoom del navegador (Ctrl +, Ctrl -) para ajustar el tamaño.

🔢 Pulsa en los números de la derecha para cambiar de nivel.

🖌️ Keke (y el resto de sprites) fueron diseñados por Arvi Teikari para el juego Baba Is You.

💘 Haz click en el Keke gigante para comenzar a jugar.

    Controles:
  • Flechas/WASD
  • R: Reiniciar
  • Z: Deshacer
